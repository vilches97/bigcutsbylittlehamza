# Big Cuts by Little Hamza

Barbería premium en Martos, Jaén. Sitio web oficial para reservas y servicios.

## Características

- Reserva de citas online
- Galería de trabajos
- Información de servicios
- Contacto y ubicación

## Desarrollo

Este proyecto está construido con React, TypeScript y Tailwind CSS.
